# Privacy Policy — Pareto Frontier for Artificial Analysis

Last updated: 2026-07-18

This privacy policy applies to the Chrome extension Pareto Frontier for Artificial Analysis.

## What the extension does

The extension adds a Pareto button to model comparison scatter plots on Artificial Analysis. When the user clicks the button, the extension reads the chart content currently rendered in the page, calculates the Pareto frontier locally, and displays the result in a popup.

## Information processed

The extension temporarily processes the points, axes, labels, and related chart content rendered on Artificial Analysis pages that match its declared host access. This processing is necessary for the extension's single purpose.

The extension does not:

- collect or retain a user account, name, email address, login, cookie, browsing history, or payment information;
- send chart content or browsing activity to a developer-operated server;
- use analytics, advertising, tracking pixels, or personalized advertising;
- sell or share information with third parties; or
- read the clipboard that was already present before the user uses the copy feature.

When the user explicitly clicks “Copiar lista” or “Copy list”, the extension writes the generated model list to the browser clipboard. This is a user-facing action and the extension does not read the existing clipboard content. The generated result is kept only in the page while the popup is open and is discarded when the popup is closed or the page is reloaded.

## Third parties

The extension does not transfer the processed chart content to third parties. Artificial Analysis operates its own website and has its own privacy policy and terms. This extension is an independent project and is not affiliated with, sponsored by, or endorsed by Artificial Analysis.

## Security

The extension has no backend and does not transmit the processed chart content. Its scripts are packaged with the extension and do not execute remote code.

## Changes to this policy

If the extension's data practices change, this policy will be updated before the change is released.
